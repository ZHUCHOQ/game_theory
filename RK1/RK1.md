# Решение задачи нахождения смешанных стратегий в антагонистической игре

## Дано
Платежная матрица игры:
$$
A = \begin{pmatrix}
19 & 7 & 3 \\
6 & 9 & 9 \\
8 & 2 & 11
\end{pmatrix}
$$

## Шаг 1. Проверка на наличие седловой точки

Найдем нижнюю и верхнюю цены игры:

**Нижняя цена игры (α) = максимум по строкам из минимумов по столбцам:**
- Строка 1: min(19, 7, 3) = 3
- Строка 2: min(6, 9, 9) = 6
- Строка 3: min(8, 2, 11) = 2
- α = max(3, 6, 2) = 6

**Верхняя цена игры (β) = минимум по столбцам из максимумов по строкам:**
- Столбец 1: max(19, 6, 8) = 19
- Столбец 2: max(7, 9, 2) = 9
- Столбец 3: max(3, 9, 11) = 11
- β = min(19, 9, 11) = 9

Так как α = 6 ≠ β = 9, седловой точки в чистых стратегиях нет. Необходимо найти решение в смешанных стратегиях.

## Шаг 2. Проверка доминирования стратегий

Проверим, есть ли доминируемые стратегии, которые можно исключить:

**Сравнение строк:**
- Строка 1 и строка 2: 19 > 6, но 7 < 9 и 3 < 9 → нет доминирования
- Строка 1 и строка 3: 19 > 8, 7 > 2, но 3 < 11 → нет доминирования
- Строка 2 и строка 3: 6 < 8, 9 > 2, 9 < 11 → нет доминирования

**Сравнение столбцов:**
- Столбец 1 и столбец 2: 19 > 7, 6 < 9, 8 > 2 → нет доминирования
- Столбец 1 и столбец 3: 19 > 3, 6 < 9, 8 < 11 → нет доминирования
- Столбец 2 и столбец 3: 7 > 3, 9 = 9, 2 < 11 → нет доминирования

Все стратегии существенны, упростить матрицу нельзя.

## Шаг 3. Решение системы уравнений для игрока 1

Пусть игрок 1 использует смешанную стратегию p = (p₁, p₂, p₃), где p₁ + p₂ + p₃ = 1, pᵢ ≥ 0.

Если все стратегии игрока 2 активны в оптимальной стратегии, то выполняются равенства:
$$
\begin{cases}
19p₁ + 6p₂ + 8p₃ = v \\
7p₁ + 9p₂ + 2p₃ = v \\
3p₁ + 9p₂ + 11p₃ = v \\
p₁ + p₂ + p₃ = 1
\end{cases}
$$

Вычтем второе уравнение из первого:
$$(19p₁ + 6p₂ + 8p₃) - (7p₁ + 9p₂ + 2p₃) = 0$$
$$12p₁ - 3p₂ + 6p₃ = 0$$
$$4p₁ - p₂ + 2p₃ = 0 \quad (1)$$

Вычтем третье уравнение из второго:
$$(7p₁ + 9p₂ + 2p₃) - (3p₁ + 9p₂ + 11p₃) = 0$$
$$4p₁ - 9p₃ = 0$$
$$p₁ = \frac{9}{4}p₃ \quad (2)$$

Подставим (2) в (1):
$$4\cdot\frac{9}{4}p₃ - p₂ + 2p₃ = 0$$
$$9p₃ - p₂ + 2p₃ = 0$$
$$p₂ = 11p₃ \quad (3)$$

Теперь используем условие нормировки p₁ + p₂ + p₃ = 1:
$$\frac{9}{4}p₃ + 11p₃ + p₃ = 1$$
Умножим обе части на 4:
$$9p₃ + 44p₃ + 4p₃ = 4$$
$$57p₃ = 4$$
$$p₃ = \frac{4}{57}$$

Найдем остальные вероятности:
$$p₂ = 11p₃ = 11\cdot\frac{4}{57} = \frac{44}{57}$$
$$p₁ = \frac{9}{4}p₃ = \frac{9}{4}\cdot\frac{4}{57} = \frac{9}{57} = \frac{3}{19}$$

Проверим сумму: p₁ + p₂ + p₃ = 3/19 + 44/57 + 4/57 = 9/57 + 44/57 + 4/57 = 57/57 = 1.

Найдем цену игры v из любого уравнения, например из первого:
$$v = 19p₁ + 6p₂ + 8p₃ = 19\cdot\frac{3}{19} + 6\cdot\frac{44}{57} + 8\cdot\frac{4}{57} = 3 + \frac{264}{57} + \frac{32}{57} = 3 + \frac{296}{57} = \frac{171}{57} + \frac{296}{57} = \frac{467}{57}$$

## Шаг 4. Решение системы уравнений для игрока 2

Пусть игрок 2 использует смешанную стратегию q = (q₁, q₂, q₃), где q₁ + q₂ + q₃ = 1, qᵢ ≥ 0.

Если все стратегии игрока 1 активны в оптимальной стратегии, то выполняются равенства:
$$
\begin{cases}
19q₁ + 7q₂ + 3q₃ = v \\
6q₁ + 9q₂ + 9q₃ = v \\
8q₁ + 2q₂ + 11q₃ = v \\
q₁ + q₂ + q₃ = 1
\end{cases}
$$

Вычтем второе уравнение из первого:
$$(19q₁ + 7q₂ + 3q₃) - (6q₁ + 9q₂ + 9q₃) = 0$$
$$13q₁ - 2q₂ - 6q₃ = 0 \quad (4)$$

Вычтем третье уравнение из второго:
$$(6q₁ + 9q₂ + 9q₃) - (8q₁ + 2q₂ + 11q₃) = 0$$
$$-2q₁ + 7q₂ - 2q₃ = 0 \quad (5)$$

Умножим уравнение (5) на 13, а уравнение (4) на 2:
$$-26q₁ + 91q₂ - 26q₃ = 0$$
$$26q₁ - 4q₂ - 12q₃ = 0$$

Сложим уравнения:
$$87q₂ - 38q₃ = 0$$
$$q₃ = \frac{87}{38}q₂ \quad (6)$$

Подставим (6) в (5):
$$-2q₁ + 7q₂ - 2\cdot\frac{87}{38}q₂ = 0$$
Умножим на 38:
$$-76q₁ + 266q₂ - 174q₂ = 0$$
$$-76q₁ + 92q₂ = 0$$
$$q₁ = \frac{92}{76}q₂ = \frac{23}{19}q₂ \quad (7)$$

Теперь используем условие нормировки q₁ + q₂ + q₃ = 1:
$$\frac{23}{19}q₂ + q₂ + \frac{87}{38}q₂ = 1$$
Приведем к общему знаменателю 38:
$$\frac{46}{38}q₂ + \frac{38}{38}q₂ + \frac{87}{38}q₂ = 1$$
$$\frac{171}{38}q₂ = 1$$
$$q₂ = \frac{38}{171}$$

Найдем остальные вероятности:
$$q₁ = \frac{23}{19}q₂ = \frac{23}{19}\cdot\frac{38}{171} = \frac{23\cdot2}{171} = \frac{46}{171}$$
$$q₃ = \frac{87}{38}q₂ = \frac{87}{38}\cdot\frac{38}{171} = \frac{87}{171}$$

Проверим сумму: q₁ + q₂ + q₃ = 46/171 + 38/171 + 87/171 = 171/171 = 1.

Проверим цену игры v:
$$v = 19q₁ + 7q₂ + 3q₃ = 19\cdot\frac{46}{171} + 7\cdot\frac{38}{171} + 3\cdot\frac{87}{171} = \frac{874}{171} + \frac{266}{171} + \frac{261}{171} = \frac{1401}{171} = \frac{467}{57}$$

## Шаг 5. Проверка оптимальности решения

**Для игрока 1:**
- Против столбца 1: 19·(3/19) + 6·(44/57) + 8·(4/57) = 3 + 264/57 + 32/57 = 467/57 = v
- Против столбца 2: 7·(3/19) + 9·(44/57) + 2·(4/57) = 21/19 + 396/57 + 8/57 = 467/57 = v
- Против столбца 3: 3·(3/19) + 9·(44/57) + 11·(4/57) = 9/19 + 396/57 + 44/57 = 467/57 = v

**Для игрока 2:**
- Против строки 1: 19·(46/171) + 7·(38/171) + 3·(87/171) = 874/171 + 266/171 + 261/171 = 1401/171 = 467/57 = v
- Против строки 2: 6·(46/171) + 9·(38/171) + 9·(87/171) = 276/171 + 342/171 + 783/171 = 1401/171 = 467/57 = v
- Против строки 3: 8·(46/171) + 2·(38/171) + 11·(87/171) = 368/171 + 76/171 + 957/171 = 1401/171 = 467/57 = v

Условия дополняющей нежесткости выполнены, решение верно.

## Шаг 6. Сравнение с альтернативными методами

- **Метод обратной матрицы**: v ≈ 8.193 (совпадает с нашим решением)
